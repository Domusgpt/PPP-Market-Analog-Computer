PPP Hypercube Control Surface

The demo runs the production transport stack. Console access remains locked; developer tooling is recorded under the PPP security policy. Activate only from trusted workstations.

Launching loads hypercube-core-webgl-framework.html into the sandboxed frame.

  • Live ingress: WebSocket + Web Serial adapters with reconnect, latency samples, and per-frame telemetry summaries.
  • Calibration workflow: Dataset builder orchestrates motion plans, parity scoring, manifest download, and narrative QA guidance.
  • Sonic exports: Spinor topology, flux continuum, lattice, and constellation payloads stream through PPP.sonicGeometry helpers.

Legacy demo assets are packaged offline in archive/legacy-2024-site.tar.gz and excluded from deployment.

Operating Notes & Safeguards

  • Console lock: DevTools trigger the PPP security overlay; sandbox keys are issued by Paul Phillips or Clear Seas Solutions.
  • Telemetry APIs: Register window.PPP_CONFIG before loading the bundle to capture recorder updates, sonic payload clones, and calibration events.
  • Dataset discipline: Run the calibration plan before exporting benchmark artifacts so manifests, parity scores, and insights ship together.
  • Testing: Node-based regression suites cover sonic geometry, live telemetry, and calibration builders. Execute npm test before distribution.
